# Анализ лояльности пользователей Яндекс Афиши

Проект представляет собой исследовательский анализ данных для выявления факторов, влияющих на повторные покупки пользователей сервиса Яндекс Афиша.

## Цели и задачи проекта¶
Цель: Провести исследовательский анализ лояльности пользователей Яндекс Афиши для выявления факторов, влияющих на повторные покупки

## Задачи:
Загрузить данные и провести их первичный осмотр.
Провести предобработку данных.
Провести исследовательский анализ данных:
Сформулировать выводы и практические рекомендации для команды маркетинга по повышению лояльности пользователей.

## Общий вывод и рекомендации
Проведенный анализ лояльности пользователей Яндекс Афиши выявил ключевые закономерности, которые могут стать основой для стратегических решений команды маркетинга. Исследование охватило 287,405 транзакций 21,622 уникальных пользователей за 2024 год, данные были тщательно очищены и подготовлены для анализа.

Основной инсайт исследования заключается в том, что характеристики первого заказа пользователя (устройство, регион, жанр мероприятия, билетный оператор) оказывают минимальное влияние на вероятность его возврата. Вместо этого ключевыми факторами лояльности становятся поведенческие паттерны, формирующиеся уже после первой покупки.

Выявлены три наиболее значимых драйвера лояльности:

Частота покупок (корреляция 0.391) - пользователи с 5+ заказами совершают покупки в среднем каждые 10 дней

Оптимальный размер заказа - покупка 2-3 билетов дает максимальную лояльность (73.6%)

Стоимость заказа - вернувшиеся пользователи демонстрируют более высокую медианную стоимость заказа (496.54 ₽ против 378.87 ₽ у одноразовых)

Парадоксально, но наиболее массовые точки входа (концерты - 44% новых пользователей, мобильные устройства - 83%) показывают лишь средний уровень удержания. В то же время менее массовые сегменты демонстрируют более высокую лояльность: выставки и театр (+2-3% к среднему), десктоп-пользователи (+2.5%), отдельные регионы вроде Шанырского (+6.0%).

Наиболее уязвимыми оказались групповые покупки (5+ билетов) - их лояльность составляет всего 18.8%, что указывает на разовый характер таких заказов.

Практические рекомендации Сместить фокус с привлечения на удержание - ресурсы следует перераспределить в пользу программ, стимулирующих повторные покупки в оптимальном ритме (2-4 недели между заказами).

Разработать таргетированные предложения для семейной аудитории - сегмент покупателей 2-3 билетов (44% пользователей, 73.6% лояльности) является ядром лояльной аудитории и требует специальных программ.

Оптимизировать работу с нишевыми сегментами - выставки, театр и определенные регионы демонстрируют более высокую лояльность при меньшем объеме, что делает их стратегически важными для качественного роста.

Внедрить систему триггерных коммуникаций - напоминания и предложения через 21 день после покупки для пользователей с 1-2 заказами, чтобы увеличить их шансы перейти в категорию постоянных клиентов.

Мониторинг и анализ крупных заказов - для покупок 5+ билетов разработать отдельную стратегию конвертации разовых групповых покупок в повторные.

Таким образом, путь к росту лояльности лежит не в поиске "идеального" нового клиента, а в создании условий, при которых любой новый пользователь с высокой вероятностью станет постоянным.
